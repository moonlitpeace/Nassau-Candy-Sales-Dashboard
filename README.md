# 🍬 Nassau Candy Sales Dashboard

## 📌 Project Overview

This project is an interactive **Sales Dashboard** developed using **Python** and **Streamlit** to analyze the Nassau Candy Distributor dataset.  
The dashboard provides meaningful insights into sales performance, profit, products, regions, cities, and shipping trends through interactive visualizations.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Identify sales trends and business insights
- Build an interactive Streamlit dashboard for visualization

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit

---

## 📊 Dashboard Features

- KPI Cards (Total Sales, Profit, Orders, Customers)
- Monthly Sales Trend Analysis
- Sales by Division
- Sales by Region
- Top 10 Products
- Top 10 Cities
- Ship Mode Analysis
- Sales vs Cost Analysis
- Correlation Heatmap
- Distribution of Sales

---

## 📂 Dataset

Nassau Candy Distributor Sales Dataset
